# 훈노트 v4.5.8 테스트 체크리스트

## 메인 화면

- [ ] 메모/할 일 탭 아래에 별도의 `메모` 제목이 표시되지 않는다.
- [ ] 카테고리 필터에 `더보기` 버튼이 없다.
- [ ] 카테고리가 많으면 한 줄에서 좌우로 밀어 볼 수 있다.
- [ ] `관리` 버튼이 카테고리 필터와 같은 줄 오른쪽에 표시된다.
- [ ] 관리 버튼으로 카테고리 추가·이름 변경·삭제 창이 열린다.

## 새 메모

- [ ] 카테고리 선택 아래의 `카테고리 추가·관리` 버튼이 정상 작동한다.
- [ ] 메모 저장·수정·초안 자동 저장 기능이 이전과 동일하게 작동한다.

## 메뉴

- [ ] 톱니바퀴 아이콘과 `설정` 버튼이 표시된다.
- [ ] 설정을 누르면 카테고리 관리와 앱 설치 항목이 펼쳐진다.
- [ ] 계정 관리·데이터 관리·사용 안내 접힘 메뉴가 정상 작동한다.
- [ ] 메뉴 하단에 `v4.5.8`이 표시된다.

## PWA

- [ ] 홈 화면의 H 로고와 설치 후 앱 아이콘 디자인이 동일하다.
- [ ] 기존 설치 앱에서 업데이트 후 새 아이콘이 반영된다.
- [ ] 반영되지 않으면 앱 바로가기를 삭제한 후 다시 설치해 확인한다.
