# 훈노트 v4.5.4 실기기 테스트 체크리스트

## 배포·업데이트

- [ ] `?v=454` 주소에서 메뉴 하단 `v4.5.4`가 보인다.
- [ ] 설치 앱에 업데이트 안내가 표시되고 적용 후 정상 실행된다.
- [ ] 기존 메모·카테고리·체크리스트가 빠짐없이 보인다.

## 메인 화면·정렬

- [ ] 메모 개수 표시와 필터 버튼이 보이지 않는다.
- [ ] 카테고리 디자인과 배치가 기존과 동일하다.
- [ ] 정렬 5종이 작고 연한 텍스트와 `ㅣ` 구분선으로 보인다.
- [ ] 320px 스마트폰에서도 정렬 영역에 가로 스크롤이 생기지 않고 자연스럽게 줄바꿈된다.
- [ ] 각 정렬 항목을 누르면 목록 순서가 맞게 변경되고 선택 항목만 은은하게 강조된다.
- [ ] 검색은 제목·내용·체크리스트를 찾고 숨겨진 프로젝트 값은 검색하지 않는다.

## 작성·프로젝트 호환

- [ ] 작성 화면에 프로젝트 입력과 최근 프로젝트가 보이지 않는다.
- [ ] 메모 본문 입력 영역의 세로 높이가 충분하다.
- [ ] 새 메모 작성·수정·중요 표시·체크리스트 저장이 정상이다.
- [ ] 예전에 프로젝트가 입력된 메모도 열기·수정·백업이 정상이며 프로젝트 문구는 화면에 보이지 않는다.
- [ ] 빈 결과 화면에는 `새 메모 작성`만 있고 `전체 메모 보기`는 없다.

## 휴지통·데이터 관리

- [ ] 메뉴 상단에 별도 휴지통 버튼이 없다.
- [ ] 데이터 관리 안의 `휴지통`이 누를 수 있는 버튼으로 보인다.
- [ ] `휴지통` 버튼을 누르면 휴지통 전용 화면이 열린다.
- [ ] 휴지통 이동, 복원, 영구 삭제와 휴지통 비우기가 정상이다.

## 뒤로가기

- [ ] 메뉴에서 스마트폰 뒤로가기를 누르면 메모 메인 화면으로 돌아오고 앱이 닫히지 않는다.
- [ ] 메모 상세 보기에서 뒤로가면 상세 창만 닫힌다.
- [ ] 카테고리 관리와 계정 삭제 창에서 뒤로가면 해당 창만 닫힌다.
- [ ] 새 메모 작성창에서 뒤로가면 작성창이 닫힌다.
- [ ] 작성 내용이 있는 상태에서 뒤로가면 확인 안내가 보이고, 취소하면 작성창이 유지된다.
- [ ] 할 일 화면에서 뒤로가면 이전 앱 화면으로 돌아온다.
- [ ] 휴지통 화면에서 `메모로 돌아가기`를 누르면 메모 화면으로 간다.
- [ ] 메모 메인 화면에서 뒤로가기는 이전 웹페이지 이동 또는 앱 종료로 동작한다.

## 보안·백업 회귀

- [ ] JSON 백업·복원이 정상이고 중복 메모는 제외된다.
- [ ] 손상되거나 10MB를 초과한 백업은 거부된다.
- [ ] 로그아웃 후 다른 계정의 메모가 남지 않는다.
- [ ] 오프라인에서 앱 화면은 열리고 저장 시 온라인 연결 안내가 표시된다.
