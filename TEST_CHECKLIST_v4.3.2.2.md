# 업무노트 v4.3.2.2 테스트 체크리스트

## 화면 전환

- [ ] 오른쪽 메뉴의 휴지통을 누르면 일반 메모 목록이 아닌 전용 화면이 열린다.
- [ ] 휴지통 화면에서는 새 메모, 검색, 카테고리, 프로젝트 필터가 보이지 않는다.
- [ ] 메모로 돌아가기를 누르면 일반 메모 화면으로 돌아간다.
- [ ] 상단 메모/할 일 탭을 눌러 정상적으로 화면을 전환할 수 있다.

## 휴지통 기능

- [ ] 일반 메모를 삭제하면 휴지통 전용 화면에 나타난다.
- [ ] 복원을 누르면 해당 메모만 일반 메모로 복원된다.
- [ ] 영구 삭제는 확인 후 해당 메모만 완전히 삭제한다.
- [ ] 휴지통 비우기는 확인 후 모든 삭제 메모를 완전히 삭제한다.
- [ ] 휴지통이 비면 빈 상태 안내가 표시된다.
- [ ] 삭제 메모 카드를 누르면 상세 화면이 열린다.
- [ ] 상세 화면의 복구하기와 완전 삭제가 정상 작동한다.

## 회귀 테스트

- [ ] 일반 메모 목록에는 휴지통 메모가 나타나지 않는다.
- [ ] 메모 작성·수정·삭제가 정상 작동한다.
- [ ] 할 일 화면과 체크리스트가 정상 작동한다.
- [ ] 백업·복원과 클라우드 동기화가 정상 작동한다.
- [ ] PC와 스마트폰에서 휴지통 화면이 깨지지 않는다.
