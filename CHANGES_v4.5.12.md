# 훈노트 v4.5.12 변경 사항

## 반영 항목

- 상단 HOONNOTE 로고의 파란 H 심볼을 정중앙으로 재배치
- 메모 작성 화면의 기본 브라우저 카테고리 선택창을 앱 전용 커스텀 패널로 교체
- 선택 카테고리 강조, 체크 표시, 스크롤 가능한 카테고리 목록 적용
- 메뉴 메인 화면에 설정 항목을 배치
- 설정 메뉴에서는 우측 상단에 뒤로가기 화살표 표시
- 계정 관리의 로그아웃·계정 삭제 버튼을 아이콘과 설명이 포함된 카드형 UI로 정리
- 모바일 메뉴 패널 가로 폭을 화면의 80%로 조정
- PWA 정적 파일 캐시를 v4.5.12 / v=462로 갱신
