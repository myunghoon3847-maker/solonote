# 훈노트 v4.5.2 테스트 체크리스트

## 버전과 화면

- [ ] `?v=452` 주소에서 메뉴 하단에 `v4.5.2`가 표시된다.
- [ ] 상단에 `메모·할 일` 탭이 모두 표시된다.
- [ ] 상단 `할 일` 탭을 누르면 기존 할 일 화면이 열린다.

## 카테고리

- [ ] 메모 카테고리 필터에 `할 일`이 표시되지 않는다.
- [ ] 새 메모의 카테고리 선택창에 `할 일`이 표시되지 않는다.
- [ ] `중요·보관·미분류`가 연한 스카이블루로 구분된다.
- [ ] 보호 카테고리를 선택하면 진한 스카이블루로 표시된다.
- [ ] 업무·아이디어·일상과 사용자 카테고리는 기존 흰색 스타일을 유지한다.
- [ ] 카테고리 추가·이름 변경·삭제가 정상 작동한다.
- [ ] 카테고리 삭제 후 연결된 메모가 `미분류`에 표시된다.

## 회귀

- [ ] 로그인·로그아웃과 클라우드 새로고침이 정상 작동한다.
- [ ] 메모 작성·수정·삭제·복원이 정상 작동한다.
- [ ] 체크리스트와 상단 할 일 화면이 정상 작동한다.
- [ ] PC와 스마트폰에서 같은 카테고리와 메모가 표시된다.
