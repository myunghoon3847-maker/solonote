# 훈노트 v4.5.8 변경 사항

## UI·UX

- 카테고리 `더보기/접기` 버튼과 관련 동작을 제거했습니다.
- 카테고리를 한 줄 가로 스크롤 필터로 변경했습니다.
- `카테고리 관리` 버튼을 `관리`로 줄이고 필터 줄 오른쪽에 고정했습니다.
- 모바일 메모 화면에서 탭 아래에 반복 표시되던 `메모` 제목을 제거했습니다.
- 메뉴 첫 영역에 톱니바퀴 아이콘이 있는 `설정` 접힘 메뉴를 추가했습니다.
- 설정 하위 메뉴에서 카테고리 관리와 웹 앱 설치 기능을 이용할 수 있도록 구성했습니다.

## 앱 아이콘

- 헤더 홈 로고와 웹 앱 설치 아이콘이 동일한 H 심볼을 사용하도록 통일했습니다.
- 일반 아이콘, 마스커블 아이콘, 단색 아이콘, Apple Touch Icon, Play Store 원본 아이콘을 새로 생성했습니다.
- 매니페스트와 서비스워커 아이콘 경로에 `v=458`을 적용해 이전 아이콘 캐시를 갱신합니다.

## 내부 변경

- 앱·백업·계정 삭제 클라이언트 버전을 4.5.8로 변경했습니다.
- PWA 캐시 이름을 `hoonnote-v4-5-8-cache`로 변경했습니다.
