# v2.5 데이터 이전 계획

현재 v2.5 메모는 브라우저 localStorage에 저장됩니다.

v3.x에서는 Supabase DB에 저장해야 하므로 이전 과정이 필요합니다.

## 추천 이전 방식

```text
v2.5에서 백업하기
↓
JSON 파일 저장
↓
v3.x에서 로그인
↓
백업 JSON 가져오기
↓
DB에 업로드
```

## 이 방식의 장점

- 중간에 문제가 생겨도 JSON 백업 파일이 남아 있음
- 기존 localStorage 데이터를 직접 건드리지 않아 안전함
- PC와 스마트폰 간 이동도 쉬움

## v3.5에서 만들 기능

- 백업 JSON 파일 선택
- JSON 구조 확인
- memos 배열 추출
- 로그인한 사용자의 user_id로 DB 저장
- 중복 메모 방지

## 주의

v3 전환 전에는 반드시 v2.5에서 백업 파일을 먼저 저장해야 합니다.
