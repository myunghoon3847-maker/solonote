# 훈노트 v4.5.7 변경 사항

## 모바일 화면 안정화

- 앱과 인증 화면에 좌우 안전영역을 적용했습니다.
- 320·360·375·390·412·430px 화면에서 문서 폭과 화면 폭을 비교했습니다.
- 보이는 요소의 좌우 좌표를 검사해 화면 밖으로 나가는 요소가 없도록 조정했습니다.

## 카테고리

- 카테고리 버튼을 기본 한 줄로 제한했습니다.
- 한 줄을 넘을 때만 `더보기` 버튼이 나타납니다.
- 펼친 상태에서는 버튼 문구가 `접기`로 바뀝니다.
- `카테고리 관리` 버튼을 카테고리 목록 아래로 이동했습니다.
- 새 메모 작성창의 `카테고리 추가·관리` 버튼을 카테고리 패널 내부로 이동했습니다.

## 우측 메뉴

- 모바일 메뉴 폭을 약 84%로 줄여 왼쪽 바깥 영역을 충분히 남겼습니다.
- 왼쪽 바깥 영역을 누르면 메뉴가 닫힙니다.
- `계정 삭제`를 `계정 관리`의 접힘 하위 메뉴로 이동했습니다.
- 메뉴 본문과 하단 영역에 안전영역 및 독립 스크롤 처리를 적용했습니다.
- 페이지가 아래로 내려간 상태에서도 메뉴 하단 버전 영역이 화면 안에 유지됩니다.
