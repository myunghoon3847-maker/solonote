# 훈노트 v4.5.13.1 변경 사항

## 비밀번호 재설정 흐름 수정

- 비밀번호 변경 완료 후 복구 URL과 복구 상태를 즉시 정리
- 변경 완료 후 로컬 로그아웃하고 깨끗한 로그인 주소로 다시 진입
- 새 비밀번호로 로그인했을 때 재설정 화면이 다시 나타나던 문제 수정
- PASSWORD_RECOVERY 이벤트가 완료된 복구 흐름을 다시 열지 않도록 보호 상태 추가

## 재설정 이메일 요청 제한 대응

- 재설정 이메일 발송 후 60초 재요청 대기 버튼 적용
- 새로고침 후에도 남은 대기 시간을 유지
- 429 및 이메일 발송 한도를 구분해 이해하기 쉬운 안내 표시
- 여러 메일을 받은 경우 가장 최근 링크를 사용하도록 안내
