# 훈노트 v4.5.1 테스트 체크리스트

## 배포·캐시

- [ ] `?v=451` 주소에서 메뉴 하단 버전이 `v4.5.1`로 표시된다.
- [ ] 카테고리 관리 버튼이 회색 기본 버튼이나 세로 글자로 표시되지 않는다.
- [ ] 설치 앱에 업데이트 안내가 보이면 적용 후 앱이 다시 열린다.
- [ ] 새로고침 후에도 v4.4·v4.5 화면으로 되돌아가지 않는다.

## 카테고리 UI

- [ ] `카테고리` 제목 오른쪽에 파란색 `관리` 텍스트 버튼이 보인다.
- [ ] `전체·중요·업무·아이디어·일상·할 일·보관`은 같은 형태의 둥근 탭으로 보인다.
- [ ] 모바일에서 카테고리 탭만 좌우로 스크롤되고 `관리` 버튼 위치는 고정된다.
- [ ] 320px 폭에서도 글자가 세로로 쪼개지거나 화면 밖으로 넘치지 않는다.

## 관리 기능

- [ ] `관리`를 누르면 카테고리 관리창이 열린다.
- [ ] 닫기 버튼·바깥 영역·Esc 키로 관리창을 닫을 수 있다.
- [ ] 카테고리 추가·이름 변경·삭제가 정상 작동한다.
- [ ] 삭제된 카테고리의 메모가 `미분류`로 이동하고 본문은 유지된다.
- [ ] PC와 스마트폰에서 같은 카테고리 목록이 보인다.

## 기존 기능 회귀

- [ ] 기존 메모 수·본문·체크리스트가 그대로다.
- [ ] 메모 작성·수정·검색·중요·보관·휴지통이 정상 작동한다.
- [ ] 로그인·로그아웃·계정 삭제 흐름이 정상 작동한다.
- [ ] JSON 백업과 복원이 정상 작동한다.
