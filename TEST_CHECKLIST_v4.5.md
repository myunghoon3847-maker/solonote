# 훈노트 v4.5 테스트 체크리스트

## 배포·업데이트

- [ ] `?v=450` 주소에서 화면 하단 버전이 `v4.5`로 표시된다.
- [ ] 서비스워커 캐시가 `hoonnote-v4-5-cache`로 갱신된다.
- [ ] 기존 계정으로 로그인했을 때 기존 메모 수와 내용이 그대로다.
- [ ] PC와 스마트폰에서 같은 카테고리 목록이 보인다.

## 카테고리 추가

- [ ] `카테고리 관리` 창이 열린다.
- [ ] 새 카테고리를 추가하면 필터와 메모 작성 선택란에 즉시 나타난다.
- [ ] 다른 기기에서 새로고침하면 추가한 카테고리가 나타난다.
- [ ] 빈 이름, 같은 이름, 20자 초과 이름이 거부된다.
- [ ] `전체`, `중요`, `할 일`, `보관`, `휴지통`, `미분류` 이름이 거부된다.

## 이름 변경

- [ ] 카테고리 이름을 변경하면 필터와 작성 선택란이 함께 바뀐다.
- [ ] 해당 카테고리의 일반 메모와 휴지통 메모가 모두 새 이름으로 바뀐다.
- [ ] 이름 변경 뒤 기존 메모를 수정해도 충돌 경고 없이 저장된다.
- [ ] 다른 기기에서도 변경된 이름과 메모 분류가 보인다.

## 삭제·데이터 보존

- [ ] 카테고리 삭제 전 연결된 메모 수가 안내된다.
- [ ] 카테고리를 삭제해도 메모 수와 내용은 줄어들지 않는다.
- [ ] 연결된 메모가 `미분류` 필터에서 보인다.
- [ ] 휴지통에 있던 메모도 복구 후 `미분류`로 보인다.
- [ ] 마지막 카테고리는 삭제할 수 없다.

## 백업·복원·회귀

- [ ] JSON 백업 파일에 `categories` 배열이 포함된다.
- [ ] v4.5 백업을 복원하면 사용하지 않은 빈 카테고리도 복원된다.
- [ ] v4.4 이전 백업도 복원된다.
- [ ] 메모 작성·수정·휴지통·복구·영구 삭제가 정상 작동한다.
- [ ] 체크리스트, 중요 표시, 검색, 프로젝트 필터가 정상 작동한다.
- [ ] 로그아웃·재로그인과 계정 삭제 흐름이 정상 작동한다.
