# 훈노트 v4.5.7

훈노트 v4.5.7은 모바일 화면의 오른쪽 잘림, 카테고리 표시 방식, 우측 메뉴 구조와 스크롤을 정리한 UI·UX 개선 배포본입니다.

## 주요 변경

- 스마트폰 안전영역을 반영하고 화면 밖으로 나가는 요소를 차단
- 카테고리를 기본 한 줄로 표시하고, 넘칠 때만 `더보기/접기` 제공
- 카테고리 관리 버튼을 카테고리 목록 아래로 이동
- 우측 메뉴 폭을 모바일 화면의 약 84%로 축소
- 메뉴 바깥 왼쪽 영역을 누르면 메뉴가 닫히도록 유지
- 계정 삭제를 `계정 관리` 하위 메뉴로 이동
- 새 메모 작성창의 카테고리 추가·관리 기능을 카테고리 패널 안으로 이동
- 메뉴를 아래까지 스크롤했을 때 하단 영역이 잘리지 않도록 수정
- 서비스워커 및 정적 파일 캐시 버전을 `v=457`로 통일
