# SoloNote v4.0

SoloNote v4.0은 저장 버튼을 누르기 전의 작성 내용도 현재 브라우저에 자동 보관하고,
새로고침이나 앱 종료 후 다시 이어서 작성할 수 있도록 만든 초안 복구 버전입니다.

## v4.0 추가 기능

- 메모 작성·수정 내용을 약 0.7초 후 자동으로 로컬 초안 저장
- 제목, 내용, 프로젝트, 카테고리, 중요 표시, 체크리스트 보관
- 수정 중인 메모의 ID와 수정 기준 시각 보관
- 새로고침 또는 앱 재실행 후 `저장하지 않은 초안` 배너 표시
- `이어서 작성`으로 작성 화면 복구
- `초안 삭제`로 저장된 초안 제거
- 원본 메모가 삭제된 수정 초안은 새 메모 초안으로 안전하게 복구
- 저장 완료, 수정 취소, 로그아웃 시 해당 초안 정리
- 30일이 지난 오래된 초안 자동 삭제
- 로그인 사용자별로 초안을 분리하여 저장

## 중요한 저장 방식

- 정식 메모: Supabase 클라우드에 저장
- 저장 전 초안: 현재 브라우저의 localStorage에 임시 저장

초안은 다른 기기에는 동기화되지 않습니다.
정식으로 보관하려면 반드시 `저장하기` 또는 `수정 완료`를 눌러야 합니다.
