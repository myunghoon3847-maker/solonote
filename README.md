# SoloNote v3.5

SoloNote v3.5는 기존 브라우저 메모를 Supabase 클라우드로 안전하게 옮기고,
PC와 스마트폰의 동기화 상태를 확인하기 위한 버전입니다.

## v3.5 추가 기능

- 현재 브라우저의 이전 localStorage 메모 개수 표시
- 기존 브라우저 메모를 Supabase로 직접 이전
- 이미 클라우드에 있는 메모는 중복 제외
- 이전 후에도 브라우저 원본 데이터 유지
- 클라우드 새로고침 버튼
- 최근 동기화 시간 표시
- 클라우드 메모와 휴지통 개수 표시

## 안전 원칙

기존 localStorage 데이터는 자동 삭제하지 않습니다.
이전 버튼을 여러 번 눌러도 제목, 내용, 카테고리, 프로젝트, 작성일이 같은 메모는 중복 제외합니다.

## 권장 동기화 점검

1. PC에서 `PC 동기화 테스트` 메모를 작성합니다.
2. 스마트폰에서 같은 계정으로 로그인합니다.
3. `클라우드 새로고침`을 누릅니다.
4. 메모가 보이면 스마트폰에서 내용을 수정합니다.
5. PC에서 `클라우드 새로고침`을 눌러 수정 내용이 반영되는지 확인합니다.
