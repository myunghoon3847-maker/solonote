# SoloNote v3.9

SoloNote v3.9는 기능을 더 늘리기보다 실제 사용과 배포 전에 필요한 안정성과 UI를 정리한 버전입니다.

## v3.9 핵심 변경

- 작성 중인 메모가 있을 때 새로고침·창 닫기 경고
- 작성 중 로그아웃 시 확인
- 수정 취소 시 저장하지 않은 내용 확인
- 검색 지우기와 검색·필터 전체 초기화
- 현재 표시 중인 메모 개수와 필터 상태 안내
- 메모가 없을 때 바로 새 메모 작성 또는 필터 초기화 가능
- 모바일 화면에 고정 `새 메모` 버튼
- 상세 모달을 열면 배경 스크롤 차단
- 상세 모달을 닫으면 이전에 누른 메모로 포커스 복귀
- 데이터 관리 도구를 기본 접힘으로 변경
- 이전 브라우저 메모가 없으면 이전 패널 숨김
- 사용 안내 토글의 중복 JavaScript 코드 제거
- 모바일 버튼 크기와 포커스 표시 개선

## 기존 기능 유지

- Supabase 로그인과 비밀번호 재설정
- 사용자별 클라우드 메모 저장
- PC·스마트폰 동기화
- 동시 수정 충돌 감지
- 기존 메모 이전과 중복 방지
- JSON 백업·복원
- PWA 설치와 업데이트 알림

## v3.9.1 검색 화면 단순화

- 별도 `검색 지우기` 버튼 제거
- 검색창의 기본 X 버튼으로 검색어 삭제
- `필터 초기화`를 이해하기 쉬운 `전체 메모 보기`로 변경
- `전체 메모 보기`를 누르면 검색어, 카테고리, 프로젝트, 정렬을 기본 상태로 복원
