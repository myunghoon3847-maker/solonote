# SoloNote v3.6

SoloNote v3.6은 PC와 스마트폰을 오갈 때 클라우드 동기화를 더 안전하게 처리하는 안정화 버전입니다.

## v3.6 추가 기능

- 다른 탭이나 앱에서 SoloNote 화면으로 돌아오면 자동 동기화
- 인터넷 연결이 복구되면 자동 동기화
- `동기화 중`, `동기화 완료`, `오프라인`, `충돌 감지` 상태 표시
- 동시에 여러 번 메모를 불러오는 요청 방지
- 네트워크 오류 시 이미 불러온 메모 목록 유지
- 메모 편집 시작 시점의 `updated_at` 보관
- 다른 기기에서 먼저 수정한 경우 저장 충돌 감지
- 충돌 시 현재 작성 내용으로 덮어쓰기 또는 서버 최신 내용 불러오기 선택
- 체크리스트 충돌 시 최신 내용을 불러오고 다시 시도하도록 안내
- 수동 `클라우드 새로고침` 유지

## 충돌 처리 방식

1. 메모 수정 화면을 열 때 서버 수정 시각을 기억합니다.
2. 저장 직전에 Supabase의 현재 수정 시각을 확인합니다.
3. 수정 시각이 달라졌다면 다른 기기에서 먼저 바뀐 것으로 판단합니다.
4. 사용자가 덮어쓰기 또는 최신 서버 내용 불러오기를 선택합니다.

## 오프라인 동작

오프라인에서는 기존에 불러온 메모를 볼 수 있지만 새 저장과 수정은 차단됩니다.
인터넷이 다시 연결되면 자동으로 Supabase의 최신 메모를 불러옵니다.
