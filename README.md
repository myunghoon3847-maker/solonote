# SoloNote v4.1

SoloNote v4.1은 여러 메모에 흩어진 체크리스트를 한 화면에서 관리하는
`통합 할 일` 기능을 추가한 버전입니다.

## v4.1 추가 기능

- 모든 일반 메모의 체크리스트를 한곳에 모아 표시
- 미완료 할 일 개수를 항상 확인
- `미완료`와 `완료 포함` 화면 전환
- 통합 할 일 화면에서 바로 완료·미완료 처리
- 원본 메모 제목을 눌러 상세 화면 열기
- 프로젝트, 카테고리, 수정 날짜 표시
- 완료된 항목은 취소선과 완료 표시
- 다른 기기에서 먼저 수정된 경우 기존 충돌 감지 유지
- 휴지통 메모의 체크리스트는 통합 할 일에서 제외
- 별도 Supabase 컬럼 없이 기존 `tasks` 데이터 활용

## 사용 방법

1. 저장된 메모 영역에서 `통합 할 일`을 찾습니다.
2. `할 일 보기`를 누릅니다.
3. 체크박스 모양의 항목을 누르면 완료 상태가 저장됩니다.
4. 오른쪽 또는 아래쪽의 메모 제목을 누르면 원본 메모가 열립니다.
5. `완료 포함`을 누르면 완료한 체크리스트도 함께 볼 수 있습니다.
