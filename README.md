# SoloNote

SoloNote는 1인 개발자, 프리랜서, 창작자, 개인 사업자가 업무 메모, 아이디어, 할 일, 프로젝트 메모를 정리하기 위한 개인 업무노트 웹앱입니다.

## 현재 버전

v2.5 개인용 MVP 안정판

## 주요 기능

- 메모 작성, 저장, 상세 보기, 수정, 삭제
- 카테고리 필터
- 검색
- 중요 메모 표시
- 휴지통, 복구, 완전 삭제
- 작성 영역 접기 / 펼치기
- 백업 / 복원
- PWA 설치 지원
- 정렬
- 체크리스트
- 프로젝트별 메모 묶기
- 최근 프로젝트 빠른 선택
- 데이터 관리
- 앱 정보 / 사용 안내

## 저장 방식

현재 SoloNote는 서버나 데이터베이스 없이 브라우저 localStorage에 메모를 저장합니다.

따라서 다른 브라우저, 다른 스마트폰, 다른 PC에서는 메모가 자동으로 보이지 않습니다.
중요한 메모는 백업하기 기능으로 JSON 파일을 따로 보관하는 것을 권장합니다.
