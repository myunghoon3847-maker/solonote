# 훈노트 v4.5.13.2

훈노트 v4.5.13.2는 v4.5.13.1의 비밀번호 재설정 긴급 수정을 유지하면서, 메모 작성 UI를 이전의 메인 화면 내부 펼침형 패널로 복원한 버전입니다.

## 주요 변경

- 독립된 메모 작성 화면 제거
- 메인 화면의 검색 영역 위에 작성 패널 표시
- 데스크톱 `+ 새 메모` 버튼으로 작성 패널 열기·닫기
- 모바일 플로팅 `+ 새 메모` 버튼으로 작성 패널 열기
- 메모 수정과 자동 초안 복구도 같은 작성 패널에서 처리
- 기존 카테고리 전용 선택 패널과 체크리스트 UI 유지
- 비밀번호 재설정 단일 흐름과 이메일 요청 제한 안내 유지

## 자동 검사

- 보안·정적 회귀 테스트 8/8 통과
- 모의 Supabase 핵심 기능 검사 17/17 통과
