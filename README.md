# SoloNote v3.8

SoloNote v3.8은 웹사이트를 스마트폰과 PC에 설치하고,
새 버전을 안전하게 적용할 수 있도록 PWA 사용성을 보강한 버전입니다.

## v3.8 추가 기능

- 설치 가능한 브라우저에서 헤더에 `앱 설치` 버튼 표시
- 설치 완료 여부 안내
- 안드로이드, 아이폰, PC별 설치 방법 안내
- 새 서비스 워커가 준비되면 업데이트 배너 표시
- `지금 업데이트`를 누르면 새 버전 적용 후 자동 새로고침
- `나중에`를 누르면 현재 작업을 유지
- 앱으로 설치된 상태에서는 설치 버튼 자동 숨김
- 화면으로 돌아올 때 새 버전 확인
- PWA 전용 `pwa.js`로 설치와 업데이트 기능 분리

## 설치 방법

### 안드로이드 Chrome

SoloNote 접속 후 표시되는 `앱 설치` 버튼을 누릅니다.
버튼이 보이지 않으면 Chrome 메뉴에서 `앱 설치` 또는 `홈 화면에 추가`를 선택합니다.

### 아이폰 Safari

Safari 공유 버튼을 누르고 `홈 화면에 추가`를 선택합니다.

### PC Chrome·Edge

주소창 오른쪽의 설치 아이콘 또는 브라우저 메뉴의 앱 설치 항목을 사용합니다.
