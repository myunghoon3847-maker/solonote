# SoloNote v4.2

SoloNote v4.2는 새 기능을 추가하는 대신 기존 기능을 더 단순하게 찾고 사용할 수 있도록
화면 구조와 CSS를 전면 정리한 UI/UX 개편 버전입니다.

## 핵심 변경

- 메모 화면과 할 일 화면을 상단 탭으로 분리
- 메모 화면에서는 작성, 검색, 목록만 우선 노출
- 프로젝트, 정렬, 카테고리는 `필터` 안으로 이동
- 백업, 복원, 설치, 로그아웃, 데이터 관리, 사용 안내는 우측 메뉴로 이동
- PC와 스마트폰 버튼 크기 체계 재정리
- 메인 버튼, 일반 버튼, 텍스트 버튼 역할 구분
- 통합 할 일은 왼쪽 체크 영역만 완료 처리
- 할 일의 오른쪽 전체 영역은 원본 메모 열기
- 상세 메모 체크리스트도 체크 네모만 완료 처리
- 메모 카드를 한 줄 목록 형태로 단순화
- 이전 버전에서 누적된 CSS를 폐기하고 하나의 디자인 시스템으로 재작성
- 기존 로그인, 동기화, 백업, 초안 복구, PWA 기능 유지

## 화면 구조

```text
SoloNote                 동기화 · 메뉴
          [ 메모 | 할 일 ]

메모 화면
- 새 메모
- 검색
- 필터
- 메모 목록

할 일 화면
- 미완료 / 완료 포함
- 체크리스트 목록

메뉴
- 설치, 백업, 복원, 로그아웃
- 데이터 관리
- 사용 안내
```
