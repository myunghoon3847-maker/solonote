# 훈노트 v4.5.11

훈노트 v4.5.11은 상단 브랜드 로고와 메뉴 사용성을 정리한 모바일 UI·UX 수정 버전입니다.

## 이번 버전 변경

- Prompt Maker 예시 구성을 참고한 `H + HOONNOTE` 워드마크 적용
- 첨부한 회색 톱니바퀴 이미지를 설정 아이콘으로 적용
- 메뉴 패널 폭을 기존보다 최대 50px 확대
- 빠른 작업에서 로그아웃 제거
- 로그아웃은 `설정 → 계정 관리` 안에 유지
- 데이터 관리에서 `휴지통 비우기` 제거
- 휴지통 항목 오른쪽에 `열기` 표시 추가

## 배포

1. 이 폴더 안의 파일 전체를 GitHub 저장소 배포 경로에 덮어씁니다.
2. 일부 파일만 교체하지 말고 전체 파일을 반영합니다.
3. 배포 완료 후 `?v=461`을 붙여 접속합니다.
4. 메뉴 하단에서 `v4.5.11`을 확인합니다.
5. 설치 앱에 업데이트가 나타나면 적용한 뒤 앱을 완전히 종료하고 다시 실행합니다.
