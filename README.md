# SoloNote v1

1인 업무노트 / 아이디어노트 웹앱 1차 버전입니다.

## 실행 방법

1. 압축을 풉니다.
2. `index.html` 파일을 더블 클릭합니다.
3. 브라우저에서 메모를 작성하고 저장합니다.

## 포함 기능

- 메모 작성
- 메모 저장
- 메모 목록 보기
- 메모 상세 보기
- 메모 수정
- 메모 삭제
- 카테고리 필터
- 검색
- 브라우저 localStorage 저장
- 스마트폰 화면 대응

## 파일 구조

```text
solonote_v1/
├─ index.html
├─ css/
│  └─ style.css
└─ js/
   ├─ storage.js
   ├─ ui.js
   └─ app.js
```

## 주의

현재 버전은 서버 없이 브라우저에 저장됩니다.
브라우저 캐시/사이트 데이터를 삭제하면 메모가 사라질 수 있습니다.


## v1.1 수정 사항

- 스마트폰 화면에서 상단 헤더 오른쪽 간격이 맞지 않는 문제 수정
- 모바일 가로 넘침 방지
- 헤더, 패널, 레이아웃 폭을 100% 기준으로 정리


## v1.2 수정 사항

- 모바일 기준 CSS 구조로 재정리
- 오른쪽으로 화면이 밀리거나 입력창이 잘리는 문제 방지
- 상단 `0개의 메모` 박스를 작은 배지 형태로 수정
- 모바일에서 헤더, 입력폼, 패널 폭을 더 안정적으로 정리
- 긴 한글/영문 입력 시 레이아웃이 깨지지 않도록 줄바꿈 보강
