# SoloNote 출시 전 최종 점검표

## 필수

- [ ] PC Chrome에서 로그인, 작성, 수정, 삭제, 복구 확인
- [ ] 안드로이드 Chrome 또는 설치된 PWA에서 동일 기능 확인
- [ ] 서로 다른 두 기기에서 동기화 확인
- [ ] 비밀번호 재설정 이메일과 새 비밀번호 로그인 확인
- [ ] JSON 백업 파일을 실제로 내려받아 별도 보관
- [ ] 백업 JSON 복원 확인
- [ ] 오프라인 표시와 인터넷 재연결 확인
- [ ] Supabase RLS가 활성화되어 있는지 확인
- [ ] GitHub 저장소에 secret/service_role 키가 없는지 확인

## 권장

- [ ] 테스트용 중복 메모와 휴지통 정리
- [ ] README에 서비스 주소와 사용법 작성
- [ ] 앱 아이콘이 스마트폰 홈 화면에서 선명한지 확인
- [ ] 모바일에서 버튼과 입력창이 잘리지 않는지 확인
- [ ] 새 버전 업데이트 알림을 한 번 실제로 시험
- [ ] 안정 버전 ZIP과 GitHub 커밋을 별도 보관

## 현재 출시 범위

SoloNote v3.9는 본인이 사용하는 1인 로그인형 클라우드 업무노트 기준의 MVP입니다.
공개 회원가입, 다중 사용자 결제, 팀 공유 기능은 아직 출시 범위에 포함하지 않습니다.
