# 훈노트 v4.5.13 변경 사항

## 목적

v4.5.12에서 정리한 UI를 기준으로 핵심 기능 전체의 회귀 검수를 수행한 안정화 버전입니다. 새로운 사용자 기능은 추가하지 않았습니다.

## 검수 및 보강

- 로그인 초기 화면과 잘못된 로그인 차단 검사
- 회원가입 필수 동의 및 비밀번호 재설정 요청 검사
- 로그인 후 클라우드 초기 동기화 검사
- 카테고리 추가·이름 변경 및 메모 연동 검사
- 메모 작성·수정·중요 표시·체크리스트 저장 검사
- 검색·카테고리 필터·정렬 검사
- 할 일 완료·완료 포함 보기 검사
- 휴지통 이동·복원 검사
- JSON 백업·복원 및 중복 제외 검사
- 계정 삭제 진입과 확인 문구 검사
- 로그아웃 검사
- 자동 초안 저장·복구·삭제 검사
- 320~430px 모바일 가로 넘침 검사
- 보안·정적 회귀 테스트와 모바일 UI 테스트 갱신

- 앱 표시 버전: v4.5.13
- PWA 리소스 버전: v=463
- 서비스워커 캐시: hoonnote-v4-5-13-cache
