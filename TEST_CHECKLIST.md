# SoloNote v4.1 통합 할 일 테스트

## 기본 표시

- [ ] 저장된 메모 영역에 `통합 할 일` 카드가 보이는가?
- [ ] 남은 미완료 할 일 개수가 정확한가?
- [ ] 처음에는 내용이 접혀 있는가?
- [ ] `할 일 보기`와 `할 일 접기`가 정상 작동하는가?

## 할 일 관리

- [ ] 여러 메모의 체크리스트가 한곳에 표시되는가?
- [ ] 휴지통 메모의 체크리스트는 제외되는가?
- [ ] 할 일을 누르면 완료 상태로 저장되는가?
- [ ] 완료된 할 일을 다시 누르면 미완료로 돌아오는가?
- [ ] 완료 처리 후 남은 개수가 바로 갱신되는가?

## 표시 범위

- [ ] `미완료`에서는 완료하지 않은 항목만 보이는가?
- [ ] `완료 포함`에서는 모든 체크리스트가 보이는가?
- [ ] 완료 항목에 체크 표시와 취소선이 보이는가?

## 원본 메모

- [ ] 메모 제목을 누르면 해당 메모 상세창이 열리는가?
- [ ] 프로젝트와 카테고리가 올바르게 표시되는가?
- [ ] 상세창의 체크리스트 상태도 동일하게 반영되는가?

## 동기화

- [ ] PC에서 완료 처리한 내용이 스마트폰에서 동기화되는가?
- [ ] 다른 기기에서 먼저 수정한 경우 충돌 안내가 나오는가?
- [ ] 오프라인에서 체크하면 저장 불가 안내가 나오는가?

## 기존 기능

- [ ] 초안 자동 저장과 복구가 정상인가?
- [ ] 메모 작성·수정·삭제·복구가 정상인가?
- [ ] 검색과 전체 메모 보기가 정상인가?
- [ ] PWA 설치와 업데이트가 정상인가?


## v4.1.1 체크 영역 확인

- [ ] 체크 네모를 누르면 완료·미완료 상태가 바뀌는가?
- [ ] 할 일 문구를 눌러도 체크 상태가 바뀌지 않는가?
- [ ] 체크 영역이 문구 전체로 넓게 잡히지 않는가?
- [ ] 메모 제목을 누르면 원본 메모가 정상적으로 열리는가?
- [ ] 모바일에서도 체크 네모를 누르기 어렵지 않은가?
