# SoloNote v3.7 비밀번호 재설정 테스트

## 재설정 이메일

- [ ] 로그인 화면에 `비밀번호를 잊으셨나요?`가 보이는가?
- [ ] 버튼을 누르면 재설정 이메일 입력 화면이 열리는가?
- [ ] 로그인 이메일을 먼저 입력했다면 재설정 화면에 자동 입력되는가?
- [ ] 재설정 이메일 보내기 버튼이 중복 클릭되지 않는가?
- [ ] 발송 완료 안내가 보이는가?
- [ ] 이메일의 스팸함도 확인했는가?

## 새 비밀번호 설정

- [ ] 이메일 링크를 누르면 SoloNote의 `새 비밀번호 설정` 화면이 열리는가?
- [ ] 재설정 중에는 메모 앱이 보이지 않는가?
- [ ] 8자 미만 비밀번호가 차단되는가?
- [ ] 두 비밀번호가 다르면 안내되는가?
- [ ] 일치하는 새 비밀번호를 저장할 수 있는가?
- [ ] 완료 후 로그인 화면으로 이동하는가?
- [ ] 새 비밀번호로 로그인되는가?
- [ ] 기존 비밀번호로는 로그인되지 않는가?

## 오류 처리

- [ ] 만료된 링크를 열면 새 링크 요청 안내가 나오는가?
- [ ] 인터넷을 끊고 요청하면 연결 오류 안내가 나오는가?
- [ ] 로그인·로그아웃과 기존 메모 기능이 정상인가?
