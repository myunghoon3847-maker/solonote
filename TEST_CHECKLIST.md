# SoloNote v4.3.2 회원가입 테스트

## 회원가입 화면

- [ ] 로그인 화면에 `처음이신가요? 회원가입` 버튼이 보이는가?
- [ ] 회원가입 버튼을 누르면 이메일·비밀번호·비밀번호 확인 입력란이 보이는가?
- [ ] `이미 계정이 있나요? 로그인`을 누르면 로그인 화면으로 돌아오는가?
- [ ] 스마트폰에서 입력창과 버튼이 잘리지 않는가?

## 입력 검사

- [ ] 이메일을 입력하지 않으면 가입이 진행되지 않는가?
- [ ] 이메일 형식이 잘못되면 브라우저 검사가 동작하는가?
- [ ] 비밀번호가 8자 미만이면 안내가 표시되는가?
- [ ] 두 비밀번호가 다르면 안내가 표시되는가?
- [ ] 가입 처리 중 버튼이 비활성화되는가?

## 이메일 인증 사용 시

- [ ] 새 이메일로 가입하면 인증 이메일이 도착하는가?
- [ ] 가입 후 로그인 화면에 인증 안내가 표시되는가?
- [ ] 인증 전 로그인하면 인증 필요 안내가 표시되는가?
- [ ] 인증 링크를 누르면 SoloNote 주소로 돌아오는가?
- [ ] 인증 링크가 새 비밀번호 설정 화면으로 잘못 열리지 않는가?
- [ ] 인증 완료 후 로그인할 수 있는가?

## 기존 계정과 오류

- [ ] 이미 가입된 이메일로 요청해도 민감한 계정 정보가 과도하게 노출되지 않는가?
- [ ] 기존 계정은 이전 비밀번호로 정상 로그인되는가?
- [ ] 비밀번호 재설정 흐름이 그대로 작동하는가?
- [ ] 네트워크 오류 시 이해하기 쉬운 안내가 표시되는가?

## 기존 기능 회귀 테스트

- [ ] 메모 작성·수정·삭제·복원이 정상인가?
- [ ] 검색과 필터가 정상인가?
- [ ] 체크리스트와 통합 할 일이 정상인가?
- [ ] PC와 스마트폰 동기화가 정상인가?
- [ ] 백업·복원이 정상인가?
- [ ] 초안 자동 저장·복구가 정상인가?
- [ ] PWA 설치와 업데이트가 정상인가?
