# SoloNote v4.0 초안 자동 저장·복구 테스트

## 자동 저장

- [ ] 새 메모 작성창에 제목이나 내용을 입력하면 `초안 저장 준비 중`이 표시되는가?
- [ ] 잠시 후 `초안 자동 저장됨`과 시간이 표시되는가?
- [ ] 프로젝트, 카테고리, 중요 표시와 체크리스트도 초안에 포함되는가?
- [ ] 아무 내용도 입력하지 않으면 불필요한 초안이 만들어지지 않는가?

## 새로고침 복구

1. 새 메모를 작성하지만 저장 버튼은 누르지 않습니다.
2. 초안 자동 저장 완료를 확인합니다.
3. 브라우저를 새로고침합니다.
4. 브라우저의 이탈 경고가 나오면 새로고침을 계속합니다.

- [ ] 다시 접속했을 때 `저장하지 않은 초안이 있습니다` 배너가 보이는가?
- [ ] `이어서 작성`을 누르면 제목, 내용, 프로젝트와 체크리스트가 복구되는가?
- [ ] 복구 후 정식 저장이 정상적으로 되는가?
- [ ] 정식 저장 후 다시 새로고침해도 복구 배너가 나타나지 않는가?

## 수정 초안

- [ ] 기존 메모를 수정하다 새로고침하면 수정 초안이 복구되는가?
- [ ] 다른 기기에서 원본이 변경됐다면 기존 충돌 경고가 정상적으로 작동하는가?
- [ ] 원본 메모를 삭제한 뒤 수정 초안을 복구하면 새 메모 작성 상태가 되는가?

## 초안 삭제

- [ ] 복구 배너에서 `초안 삭제`를 누르면 확인 창이 나오는가?
- [ ] 삭제 후 다시 새로고침해도 복구 배너가 나타나지 않는가?
- [ ] 수정 취소 후 초안이 남지 않는가?
- [ ] 로그아웃을 완료한 후 해당 초안이 남지 않는가?
