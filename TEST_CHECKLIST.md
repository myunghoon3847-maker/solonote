# SoloNote v4.2 UI/UX 개편 테스트

## 첫 화면

- [ ] 로그인 후 메모, 검색, 새 메모 버튼이 먼저 보이는가?
- [ ] 백업, 복원, 데이터 관리, 사용 안내가 메인 화면에서 사라졌는가?
- [ ] 불필요하게 큰 버튼이나 긴 설명이 줄어들었는가?

## 메모와 할 일 탭

- [ ] `메모` 탭에서 메모 목록이 보이는가?
- [ ] `할 일` 탭에서 통합 체크리스트가 보이는가?
- [ ] 할 일 탭의 숫자가 남은 미완료 개수와 일치하는가?
- [ ] 스마트폰에서 할 일 화면일 때 새 메모 버튼이 숨겨지는가?
- [ ] 다시 메모 화면으로 돌아오면 새 메모 버튼이 보이는가?

## 검색과 필터

- [ ] 검색창은 항상 보이는가?
- [ ] 필터 버튼을 눌렀을 때만 카테고리, 프로젝트, 정렬이 보이는가?
- [ ] 적용된 필터 개수가 필터 버튼에 표시되는가?
- [ ] `전체 메모 보기`로 검색과 필터를 모두 초기화할 수 있는가?

## 할 일 터치 영역

- [ ] 왼쪽 체크 영역만 누르면 완료 상태가 바뀌는가?
- [ ] 할 일 문구와 오른쪽 영역을 누르면 원본 메모가 열리는가?
- [ ] 체크 영역이 지나치게 넓거나 좁지 않은가?
- [ ] PC와 스마트폰 모두 원본 메모 영역이 충분히 넓은가?
- [ ] 상세 메모에서도 체크 네모만 완료 처리되는가?

## 우측 메뉴

- [ ] 메뉴 버튼으로 우측 메뉴가 열리고 닫히는가?
- [ ] 배경을 누르거나 ESC 키를 누르면 메뉴가 닫히는가?
- [ ] 백업, 복원, 앱 설치, 로그아웃이 정상인가?
- [ ] 데이터 관리와 사용 안내가 접힌 상태로 시작하는가?
- [ ] 데이터 관리의 기존 기능이 모두 작동하는가?

## 작성과 기존 기능

- [ ] 새 메모 작성, 수정, 삭제, 복구가 정상인가?
- [ ] 초안 자동 저장과 복구가 정상인가?
- [ ] PC와 스마트폰 동기화가 정상인가?
- [ ] 비밀번호 재설정이 정상인가?
- [ ] PWA 설치와 업데이트가 정상인가?
