# SoloNote v3.8 PWA 설치·업데이트 테스트

## 설치

- [ ] 일반 브라우저에서 SoloNote가 정상 실행되는가?
- [ ] 설치 조건이 충족된 Chrome·Edge에서 `앱 설치` 버튼이 보이는가?
- [ ] `앱 설치` 버튼을 누르면 브라우저 설치 창이 열리는가?
- [ ] 설치 후 바탕화면 또는 홈 화면에 SoloNote 아이콘이 생기는가?
- [ ] 설치된 앱을 열면 주소창 없이 앱 형태로 실행되는가?
- [ ] 설치된 상태에서는 `앱 설치` 버튼이 숨겨지는가?
- [ ] 아이폰 안내에 `공유 → 홈 화면에 추가`가 표시되는가?

## 설치 안내

- [ ] 앱 정보의 `설치 방법 보기` 버튼이 작동하는가?
- [ ] 다시 누르면 설치 방법이 접히는가?
- [ ] 스마트폰 화면에서 안내가 잘리지 않는가?

## 업데이트

1. 먼저 v3.8을 GitHub에 올리고 한 번 접속합니다.
2. 이후 파일의 캐시 버전 또는 내용을 바꾼 다음 다시 배포해야 실제 새 버전 알림을 시험할 수 있습니다.

- [ ] 새 서비스 워커가 준비되면 업데이트 배너가 표시되는가?
- [ ] `나중에`를 누르면 배너가 닫히는가?
- [ ] `지금 업데이트`를 누르면 새 버전으로 새로고침되는가?
- [ ] 업데이트 후 기존 로그인과 메모가 유지되는가?

## 기존 기능

- [ ] 로그인·로그아웃이 정상인가?
- [ ] 비밀번호 재설정 화면이 정상인가?
- [ ] 클라우드 메모 저장과 자동 동기화가 정상인가?
- [ ] 기존 메모 이전과 중복 방지가 정상인가?
