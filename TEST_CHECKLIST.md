- [ ] 짧은 시간에 화면 복귀 이벤트가 여러 번 생겨도 목록이 흔들리지 않는가?

## 동시 수정 충돌

1. PC와 스마트폰에서 같은 메모의 수정 화면을 엽니다.
2. PC에서 먼저 수정하고 저장합니다.
3. 스마트폰에서 열린 이전 내용을 수정하고 저장합니다.

- [ ] 스마트폰에서 충돌 경고가 표시되는가?
- [ ] `확인`을 누르면 스마트폰의 현재 작성 내용으로 덮어쓰는가?
- [ ] `취소`를 누르면 PC에서 저장한 최신 서버 내용을 불러오는가?
- [ ] 체크리스트가 다른 기기에서 먼저 변경되면 최신 내용을 불러오라는 안내가 나오는가?
