# SoloNote v3.9 안정화 테스트

## 작성 중 이탈 방지

- [ ] 새 메모를 작성하다 브라우저를 새로고침하면 경고가 나오는가?
- [ ] 작성 중 로그아웃하면 확인 창이 나오는가?
- [ ] 수정 중 `수정 취소`를 누르면 확인 창이 나오는가?
- [ ] 저장 완료 후에는 불필요한 이탈 경고가 나오지 않는가?

## 검색과 필터

- [ ] 검색어를 입력하면 `검색 지우기` 버튼이 보이는가?
- [ ] 검색 지우기를 누르면 전체 검색 결과로 돌아오는가?
- [ ] 카테고리·프로젝트·정렬을 바꾸면 필터 요약이 갱신되는가?
- [ ] `필터 초기화`를 누르면 전체·최근 수정순으로 돌아오는가?
- [ ] 결과가 없을 때 새 메모 작성과 필터 초기화 버튼이 작동하는가?

## 모바일

- [ ] 모바일 하단에 `새 메모` 버튼이 보이는가?
- [ ] 버튼을 누르면 작성창으로 이동하는가?
- [ ] 작성창이 열리면 하단 새 메모 버튼이 숨겨지는가?
- [ ] 주요 버튼 높이가 작지 않고 누르기 편한가?

## 상세 화면

- [ ] 메모 상세창을 열면 배경이 스크롤되지 않는가?
- [ ] 상세창을 닫으면 이전에 누른 메모로 포커스가 돌아가는가?
- [ ] ESC 키와 배경 클릭으로 상세창이 닫히는가?

## 데이터 관리와 안내

- [ ] 데이터 관리가 처음에는 접혀 있는가?
- [ ] `관리 도구 보기`를 누르면 백업·이전·삭제 도구가 보이는가?
- [ ] 기존 브라우저 메모가 없으면 이전 패널이 숨겨지는가?
- [ ] 사용 안내 버튼이 한 번 누를 때 한 번만 열리고 닫히는가?

## 회귀 테스트

- [ ] 로그인·로그아웃과 비밀번호 재설정이 정상인가?
- [ ] 새 메모 저장·수정·삭제·복구가 정상인가?
- [ ] 자동 동기화와 충돌 감지가 정상인가?
- [ ] PWA 설치와 업데이트 기능이 정상인가?
