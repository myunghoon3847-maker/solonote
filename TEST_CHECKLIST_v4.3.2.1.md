# 업무노트 v4.3.2.1 테스트 체크리스트

## 메인 화면
- [ ] 검색창 아래에 전체·중요·업무·아이디어·할 일·일상·보관이 보인다.
- [ ] 프로젝트와 휴지통이 카테고리 탭에 보이지 않는다.
- [ ] 일상 카테고리로 메모를 저장하고 다시 열 수 있다.
- [ ] 프로젝트 입력란과 프로젝트 필터는 정상 작동한다.

## 기존 데이터
- [ ] 기존 프로젝트 카테고리 메모가 업무로 표시된다.
- [ ] 해당 메모의 프로젝트명 필드는 유지된다.

## 체크리스트
- [ ] 삭제 버튼이 충분한 크기와 빨간 보조 스타일로 보인다.
- [ ] 삭제 버튼을 누르면 선택한 항목만 제거된다.
- [ ] 메모 저장 후 다시 열어도 삭제된 항목이 돌아오지 않는다.

## 우측 메뉴
- [ ] 메뉴가 오른쪽에서 부드럽게 열린다.
- [ ] 배경이 자연스럽게 어두워진다.
- [ ] 닫기·배경·ESC로 메뉴가 부드럽게 닫힌다.
- [ ] 메뉴 최상단에 휴지통과 개수가 보인다.
- [ ] 휴지통을 누르면 휴지통 메모 목록으로 이동한다.
- [ ] 계정 관리와 계정 삭제 버튼이 메뉴 상단에서 보인다.

## 앱 이름
- [ ] 로그인 화면, 앱 상단, 메뉴 하단에 업무노트가 표시된다.
- [ ] 설치 앱 이름이 업무노트로 표시된다.
