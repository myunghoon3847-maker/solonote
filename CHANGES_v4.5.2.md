# 훈노트 v4.5.2 변경 기록

## 화면

- 상단 `메모·할 일` 탭에서 `할 일` 화면을 유지했습니다.
- 메모 작성 카테고리 선택과 메모 카테고리 필터에서 `할 일`을 제거했습니다.
- `중요·보관·미분류`를 연한 스카이블루 보호 카테고리로 표시했습니다.
- 보호 카테고리가 선택되면 진한 스카이블루로 표시합니다.
- 카테고리 관리창에 보호 카테고리는 삭제할 수 없다는 안내를 추가했습니다.

## 데이터 호환

- 기존 메모의 카테고리 값이 `할 일`이면 앱에서 `미분류`로 인식합니다.
- 메모 내용, 체크리스트, 프로젝트, 중요 표시와 날짜는 변경하지 않습니다.
- Supabase 테이블과 함수는 변경하지 않아 추가 SQL이 없습니다.

- 정적 자원: `v=452`
- 서비스 워커 캐시: `hoonnote-v4-5-2-cache`
