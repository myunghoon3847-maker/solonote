# 훈노트 v4.5.13.1 검수 보고서

## 수정 결과

- 비밀번호 재설정 완료 후 새 비밀번호로 로그인하면 앱 메인 화면으로 진입
- 재설정 화면이 두 번째로 나타나지 않음
- 재설정 이메일 연속 요청은 60초 동안 버튼에서 차단
- Supabase 이메일 발송 한도 오류는 최대 1시간 대기 안내로 구분

## 자동 검사

- 인증 복구 전용 시나리오 통과
- 보안·정적 테스트 통과

- 320~430px 모바일 UI 검사 통과

## 외부 제한

Supabase 기본 이메일 서비스의 프로젝트 전체 발송 한도는 프런트엔드 코드로 해제할 수 없습니다. 현재 제한 상태라면 한도가 회복될 때까지 기다리거나 운영 전 사용자 지정 SMTP를 연결해야 합니다.
