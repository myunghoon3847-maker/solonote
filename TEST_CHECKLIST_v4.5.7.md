# 훈노트 v4.5.7 테스트 체크리스트

## 모바일 화면

- [ ] 320px 화면에서 오른쪽이 잘리지 않는다.
- [ ] 360px 화면에서 오른쪽이 잘리지 않는다.
- [ ] 390px 화면에서 오른쪽이 잘리지 않는다.
- [ ] 가로로 화면이 움직이지 않는다.
- [ ] 상단 로고, 동기화 상태, 메뉴 버튼이 화면 안에 있다.

## 카테고리

- [ ] 카테고리가 기본 한 줄로 표시된다.
- [ ] 카테고리가 많으면 `더보기`가 표시된다.
- [ ] `더보기`를 누르면 전체 카테고리가 펼쳐진다.
- [ ] 펼친 상태에서 `접기`를 누르면 한 줄로 돌아간다.
- [ ] `카테고리 관리` 버튼이 카테고리 목록 아래에 있다.
- [ ] 카테고리 추가·이름 변경·삭제가 정상 작동한다.

## 새 메모 작성

- [ ] 카테고리 선택 영역이 하나의 패널로 보인다.
- [ ] `카테고리 추가·관리` 버튼이 카테고리 패널 안에 있다.
- [ ] 버튼을 누르면 카테고리 관리창이 열린다.
- [ ] 메모 저장과 수정이 정상 작동한다.

## 우측 메뉴

- [ ] 메뉴가 화면 전체 폭을 가리지 않는다.
- [ ] 왼쪽 바깥 영역을 누르면 메뉴가 닫힌다.
- [ ] `계정 삭제`가 처음부터 노출되지 않는다.
- [ ] `계정 관리 → 열기`를 눌러야 계정 삭제가 나타난다.
- [ ] 메뉴를 끝까지 내려도 하단 `훈노트 / v4.5.7`이 보인다.
- [ ] 페이지를 아래로 내린 상태에서 메뉴를 열어도 하단이 잘리지 않는다.

## 회귀

- [ ] 로그인·로그아웃이 정상 작동한다.
- [ ] 메모·할 일·휴지통 화면 전환이 정상 작동한다.
- [ ] 로고 클릭 시 메모 홈으로 이동한다.
- [ ] 백업·복원 기능이 정상 작동한다.
- [ ] 설치된 앱에서 최신 버전으로 갱신된다.
