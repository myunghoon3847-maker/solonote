# 훈노트 v4.5.1 변경 기록

## 목적

v4.5 배포 뒤 나타난 카테고리 UI 불일치와 `카테고리 관리` 클릭 무반응을 수정합니다. 카테고리 데이터 구조와 기존 메모는 변경하지 않습니다.

## 수정

- 카테고리 필터 안에 섞여 있던 관리 버튼을 별도 제목 행으로 이동
- 관리 버튼을 훈노트 공통 텍스트 버튼 디자인으로 통일
- 모바일에서는 실제 카테고리 탭만 가로 스크롤되도록 변경
- 카테고리 관리 이벤트를 앱 초기화 앞부분에서 먼저 연결
- 정적 자원 쿼리를 `v=451`로 일괄 갱신
- 서비스 워커 캐시를 `hoonnote-v4-5-1-cache`로 갱신
- 정적 자원의 오프라인 대체 시 검색 쿼리를 무시하지 않도록 변경해 버전 혼합 방지
- 서비스 워커 등록 시 HTTP 캐시를 우회해 업데이트 확인
- 훈노트와 관련된 이전 캐시만 정리하도록 범위 제한

## 변경하지 않음

- Supabase 테이블·함수·RLS 정책
- 메모·체크리스트·카테고리 데이터
- 로그인·계정 삭제 방식
- Android 패키지명과 TWA 설정

## 버전

- 화면 표시: `v4.5.1`
- 정적 자원: `v=451`
- PWA 캐시: `hoonnote-v4-5-1-cache`
- 백업 형식 표시: `4.5.1` (데이터 구조는 v4.5와 동일)
