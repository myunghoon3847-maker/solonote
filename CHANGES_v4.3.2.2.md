# 업무노트 v4.3.2.2 변경 기록

## 핵심 변경

- 휴지통을 일반 메모 필터에서 분리하고 앱 내부 전용 화면으로 구현
- 오른쪽 메뉴의 휴지통 버튼이 전용 화면으로 이동하도록 변경
- 전용 화면에 메모로 돌아가기, 삭제 메모 개수, 휴지통 비우기 추가
- 삭제된 메모 카드에 개별 복원 및 영구 삭제 버튼 추가
- 삭제된 메모 상세 보기와 기존 복원·영구 삭제 모달 동작 유지
- 일반 메모 목록은 삭제되지 않은 메모만 렌더링하도록 필터 구조 단순화
- 모바일에서 휴지통 버튼과 카드 동작 영역 최적화
- PWA 캐시 버전을 v4.3.2.2로 갱신

## 유지 사항

- 기존 메모 CRUD, 체크리스트, 할 일, 검색, 카테고리, 프로젝트, 동기화, 백업, 계정 삭제 기능 유지
- Supabase SQL 및 테이블 구조 변경 없음
