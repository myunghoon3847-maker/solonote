# 훈노트 v4.5.3 실기기 테스트 체크리스트

## 배포·업데이트

- [ ] `?v=453` 주소에서 메뉴 하단 `v4.5.3`이 보인다.
- [ ] 설치 앱에 업데이트 안내가 표시되고 적용 후 정상 실행된다.
- [ ] PC와 스마트폰에서 화면 배치가 v4.5.2와 동일하다.

## 로그인·데이터

- [ ] 로그인 후 기존 메모와 카테고리가 빠짐없이 보인다.
- [ ] 새 메모 작성, 수정, 중요 표시가 정상이다.
- [ ] 휴지통 이동, 복원, 영구 삭제가 정상이다.
- [ ] 다른 기기에서 수정 후 새로고침하면 최신 내용이 보인다.
- [ ] 상단 `할 일` 탭의 체크 완료·해제가 정상이다.

## 카테고리

- [ ] 카테고리 추가·이름 변경·삭제가 정상이다.
- [ ] 카테고리 삭제 시 메모가 삭제되지 않고 `미분류`로 이동한다.
- [ ] `중요·보관·미분류`의 보호 색상이 유지된다.

## 백업·복원

- [ ] JSON 백업 파일이 정상 다운로드된다.
- [ ] 방금 받은 백업 파일을 복원하면 기존 항목은 중복 제외된다.
- [ ] JSON이 아닌 파일은 거부된다.
- [ ] 10MB를 초과한 JSON 파일은 읽기 전에 거부된다.
- [ ] 손상된 JSON은 오류 안내 후 앱이 계속 작동한다.

## 계정·PWA

- [ ] 로그아웃 후 다른 계정의 이전 화면·메모가 남지 않는다.
- [ ] 계정 삭제 창의 백업 버튼과 취소가 정상이다.
- [ ] 실제 계정 삭제는 데이터 백업 후 감사용 테스트 계정에서만 확인한다.
- [ ] 오프라인에서 앱 화면은 열리고, 저장 시 온라인 연결 안내가 표시된다.
