# 훈노트 v4.5.9 변경 사항

## 이번 버전에서 완료한 항목

- 메인 헤더 이름을 `HOONNOTE`로 변경
- 메인 화면의 카테고리 관리 버튼 제거
- 카테고리가 화면 폭을 넘으면 다음 줄로 자연스럽게 배치
- 카테고리 가로 스크롤·더보기 구조 제거
- 할 일 화면 위쪽의 중복 `할 일` 제목과 미완료 안내 문장 제거
- 우측 메뉴의 `×` 닫기 버튼 제거
- 같은 위치에 기본 회색 톱니바퀴 설정 버튼 배치
- 설정 버튼 클릭 시 접힘 영역이 아닌 독립 설정 화면으로 이동
- 설정 화면에 `계정 관리` 버튼과 계정 삭제 하위 기능 배치
- 메모 작성 영역을 메인 목록에서 분리된 독립 화면으로 변경
- 작성 화면과 설정 화면에 뒤로가기 버튼 적용
- 브라우저·안드로이드 뒤로가기 기록과 화면 전환 흐름 유지
- PWA 캐시와 배포 버전을 v4.5.9로 갱신

## 보류 항목: 사용자 제공 로고 적용

현재 대화 파일 목록에서는 새 로고 원본이 확인되지 않아 v4.5.9에는 기존 H 로고가 유지되어 있습니다.
새 로고 PNG, SVG 또는 투명 배경 이미지를 다시 첨부하면 다음 버전에서 아래 항목을 한 번에 교체합니다.

- 메인 헤더 로고
- 파비콘
- PWA 일반 아이콘 192·512
- PWA 마스커블 아이콘 192·512
- Apple 홈 화면 아이콘
- Play Store 등록용 512 아이콘
