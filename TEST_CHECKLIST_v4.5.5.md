# 훈노트 v4.5.5 실기기 테스트 체크리스트

## 배포 확인

- [ ] `?v=455` 주소에서 메뉴 하단 `v4.5.5`가 보인다.
- [ ] 새 버전 업데이트 안내를 적용한 뒤 앱이 다시 열린다.
- [ ] 기존 메모와 체크리스트가 그대로 보인다.

## 상단 로고와 홈 이동

- [ ] 상단에 파란 `H` 심볼과 `훈노트` 이름이 깔끔하게 표시된다.
- [ ] 할 일 화면에서 로고를 누르면 메모 홈으로 이동한다.
- [ ] 휴지통에서 로고를 누르면 메모 홈으로 이동한다.
- [ ] 검색어나 카테고리를 선택한 뒤 로고를 누르면 기본 목록으로 돌아온다.
- [ ] 작성 중인 내용이 있을 때 로고를 눌러도 내용이 초안으로 보존된다.

## 휴지통

- [ ] 휴지통 화면에 `메모로 돌아가기` 버튼이 없다.
- [ ] 휴지통 비우기·복원·영구 삭제 기능은 정상 동작한다.
- [ ] Android/PWA 뒤로가기로 이전 화면으로 돌아갈 수 있다.

## 모바일 확대 방지

- [ ] 제목·내용·검색·카테고리 입력을 눌러도 화면이 자동 확대되지 않는다.
- [ ] 두 손가락 확대 동작이 제한된다.
- [ ] 320px 이상 화면에서 가로 스크롤이 생기지 않는다.
- [ ] 입력 글자와 버튼이 지나치게 크거나 잘리지 않는다.

## 작성 화면 카테고리

- [ ] 새 메모 작성 화면의 카테고리 오른쪽에 `추가·삭제`가 보인다.
- [ ] `추가·삭제`를 누르면 카테고리 관리 창이 열린다.
- [ ] 새 카테고리를 추가하면 선택 목록과 메인 카테고리에 즉시 반영된다.
- [ ] 카테고리 이름 변경이 정상 동작한다.
- [ ] 카테고리 삭제 시 연결된 메모가 `미분류`로 이동한다.
- [ ] 중요·보관·미분류 보호 정책이 유지된다.

## Android 주소 표시

- [ ] 홈 화면 설치 PWA에서는 브라우저 주소창이 보이지 않는다.
- [ ] Play 설치 앱에서 사이트 주소가 보이면 `ANDROID_ADDRESS_BAR_FIX_v4.5.5.md` 절차를 완료한다.
- [ ] Digital Asset Links 적용 후 앱을 재설치하여 주소 표시가 사라지는지 확인한다.
