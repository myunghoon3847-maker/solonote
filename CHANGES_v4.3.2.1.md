# 업무노트 v4.3.2.1 변경 기록

## UI
- 앱 표시 이름을 `업무노트`로 변경했습니다.
- 카테고리 탭을 메인 화면 검색창 아래로 이동했습니다.
- 우측 메뉴에 슬라이드 및 배경 페이드 애니메이션을 추가했습니다.
- 휴지통을 우측 메뉴 최상단에 배치했습니다.
- 계정 관리 영역을 상단으로 이동해 바로 보이도록 수정했습니다.
- 체크리스트 `삭제` 버튼의 크기, 테두리, 배경, hover 상태를 개선했습니다.

## 카테고리
- `일상`을 추가했습니다.
- `프로젝트`와 `휴지통`을 카테고리 탭에서 제거했습니다.
- 프로젝트명 입력 및 프로젝트 필터는 그대로 유지합니다.
- 기존 `프로젝트` 카테고리 데이터는 앱에서 `업무`로 정규화하고, 클라우드 데이터도 가능한 경우 자동 갱신합니다.

## 데이터베이스
- 신규 SQL 변경은 없습니다.
